1. Nombre del proyecto
FastDelivery & Sensor System (Examen Práctico Corte 1)

2. Objetivo del proyecto
Medir la precisión técnica en la sintaxis de PHP y la capacidad de modelado de datos mediante la definición de clases y propiedades con tipado estricto, sin el uso de comportamientos (métodos).

3. Problema que resuelve
El proyecto centraliza dos necesidades operativas:

Logística: Estructura de datos para el control y seguimiento de paquetes de envío.

Monitoreo: Definición de un modelo para sensores de plantas que registran lecturas y rangos técnicos.

4. Tecnologías utilizadas
Lenguaje: PHP 8.x (con tipado estricto).

Servidor Local: XAMPP / Apache.

Editor: VS Code (o el que hayas usado).

5. Conceptos aplicados
Basado en el temario de Programación Orientada a Objetos:

Definición de Clases.

Modificadores de acceso (public, private).

Tipado estricto de propiedades.

Instanciación de objetos.

Uso de clases predefinidas (DateTime).

Inclusión de archivos (require_once).

7. Instrucciones de ejecución
Copiar la carpeta del proyecto dentro del directorio htdocs de tu instalación de XAMPP.

Asegurarse de que el servidor Apache esté activo en el Panel de Control de XAMPP.

Abrir el navegador y navegar a la dirección: http://localhost/nombre_de_tu_carpeta/index.php.

8. Reflexión personal

¿Qué aprendí?
Aprendí a implementar el tipado estricto en PHP, lo cual ayuda a prevenir errores de lógica desde la definición de la clase, y a integrar clases nativas como DateTime.

¿Qué fue difícil?
Lo más retador fue cumplir con las restricciones de tiempo (30 minutos) manteniendo la estructura de carpetas src/Logistica solicitada sin olvidar la sintaxis correcta del tipado.

¿Qué mejoraría?
En un entorno real, añadiría un constructor para inicializar las propiedades de forma más limpia y métodos getters/setters para poder manipular la propiedad costoInterno que quedó privada.
